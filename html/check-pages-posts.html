The following tables summarise the actions that have been identified to references on the website:

Action categories

This gives the action classification that is applied to references

Links with the following beginnings are excluded from processing:

  • Festival website
  • https://actionclimateteignbridge.org/lookup_document.php/ - versioned documents
  • Unmigrated mapping sites

Often references include the current domain and just need this changing to a relative url, this plugin does this automatically.

Links identified as emails

These links have been identified as attempts at putting an email address into a post, even if they are in correct mailto: most machines won't be configured to respond correctly. Alternative contact-us links are proposed.

These changes are automated

Links identified as pages on newsite

These links have been identified as links to pages on newsite. Alternatives are suggested in most cases.

These changes are automated

Maps

Links to maps come from newsite/maps.html which are still being identied

Versioned media

These are documents with version numbers that are linked via https://actionclimateteignbridge.org/lookup_document.php

These are not changed by this process.

Media

These are other media which are referenced directly and need to be moved to the current media library.

These changes will be automated.

images

These are images (png, jpg, jpeg, webp) which should be moved from the their current location

These changes will be automated.
